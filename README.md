# Ascend - The Chosen One

<img width="885" height="494" alt="Screenshot 2026-07-16 153945" src="https://github.com/user-attachments/assets/4e781896-1999-49ec-9a31-e1150bc95310" />

This is a 3D FPS game that has a campaign and arcade mode. The player progresses through worlds and defeats enemies to win the campaign.
A player's stats are upgradable, and points to upgrade them are earned through finishing levels. There are 5 worlds and 10 levels each, with a boss at the 10th level.
Once the final boss is defeated, the game ends with special aspects, which I do not want to ruin, and you can either continue the game or exit.

To Play:
Movement - WASD
Looking - Mouse
Shooting - Right Click
Jump - Space Bar
Pause - Escape

Run:
Click this link and download the executable file to play it.
https://revank.itch.io/ascend-the-chosen-one


Note: This game is an .exe, and it only works on Windows.
