# Ascend - The Chosen One

This is a 3D FPS game that has a campaign and arcade mode. The player progresses through worlds and defeats enemies to win the campaign.
A player's stats are upgradable, and points to upgrade them are earned through finishing levels. There are 5 worlds and 10 levels each, with a boss at the 10th level.

To Play:
Movement - WASD
Looking - Mouse
Shooting - Right Click
Jump - Space Bar
Pause - Escape

Run:
Click this link and download the executable file to play it.



Note: This game is a .exe and it only works on Windows.
